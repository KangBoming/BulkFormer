# Data Files

The following files can be downloaded from Zenodo (https://doi.org/10.5281/zenodo.15559368)

## Using the Downloader (Recommended)

Download all data files automatically:

```bash
uv run bulkformer download data
```

Or download everything (data + model):

```bash
uv run bulkformer download all
```

See available options:
```bash
uv run bulkformer download --help
```
