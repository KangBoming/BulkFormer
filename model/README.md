# Pretrained Model

The pretrained BulkFormer can be downloaded from Zenodo (https://doi.org/10.5281/zenodo.15559368)

## Using the Downloader (Recommended)

Download the pretrained model automatically:

```bash
uv run bulkformer download model
```

Or download everything (model + data):

```bash
uv run bulkformer download all
```

See available options:
```bash
uv run bulkformer download --help
```
